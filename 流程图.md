# 3. 000：取指

当模式开关为：

```
SWC SWB SWA = 000
```

时，进入 **取指流程**。

图中取指流程还要根据状态位 **ST0** 分两种情况。

------

## 情况一：ST0 = 0

这通常表示当前还没有进入正常指令执行状态，需要先初始化 PC 或装入起始地址。

### W1

```
SBUS
LPC
SST0
STOP
SHORT
```

含义：

```
SBUS：开关数据送系统总线
LPC：把总线上的数据装入程序计数器 PC
SST0：置 ST0，使控制器进入 ST0 = 1 状态
STOP：暂停
SHORT：短周期，本机器周期只占一个 W1
```

也就是：

```
通过数据开关设置程序起始地址，并装入 PC。
```

> 注意：本拍不取指（无 LIR），第一条指令的取指交给下一周期
>（ST0=1）的自然 NOP 流程：复位后 IR=00H=NOP，NOP 分支会输出
> LIR+PCINC，从而完成初始进入程序。

------

## 情况二：ST0 = 1

这表示已经进入正常执行状态。

此时根据：

```
IR7 ~ IR4
```

进行指令译码。

例如：

### IR7~IR4 = 0000，NOP

```
LIR
PCINC
SHORT
```

含义：

```
LIR：取下一条指令
PCINC：PC 自增
SHORT：短周期执行，本指令只需要 W1
```

其他指令，例如 ADD、SUB、AND、LD、ST、JC、JZ、JMP 等，就进入之前整理过的指令执行流程。

------

# 补充后的取指流程总览

可以整理成这样：

```
CLR
 ↓
判断 SWC SWB SWA
 ↓
如果 SWC SWB SWA = 000，进入取指模式
 ↓
判断 ST0
```

## ST0 = 0

```
W1:
    SBUS
    LPC
    SST0
    STOP
    SHORT
```

> 仅一个 W1。本拍不取指，第一条指令的取指由下一周期
> （ST0=1）的自然 NOP 流程完成。

## ST0 = 1

```

```

例如：

```
IR7~IR4 = 0000，NOP:

    LIR
    PCINC
    SHORT
```

# 三、指令执行部分

根据指令寄存器高 4 位：

```text
IR7 ~ IR4
```

进行指令译码。

| IR7~IR4 | 指令 |
| ------- | ---- |
| 0000    | NOP  |
| 0001    | ADD  |
| 0010    | SUB  |
| 0011    | AND  |
| 0100    | INC  |
| 0101    | LD   |
| 0110    | ST   |
| 0111    | JC   |
| 1000    | JZ   |
| 1001    | JMP  |
| 1110    | STP  |

------

## 6. 0101：LD

LD 指令分两拍执行。

### W1



## 7. 0110：ST

### W1

```text

```

```text

LIR

```

含义：

```text
W1：把目标地址送入地址寄存器 LAR

```

------

## 8. 0111：JC

JC 是根据进位标志 C 判断是否跳转。

### C = 0，不跳转，W1

```text
LIR
PCINC
SHORT
```

### C = 1，跳转

#### W1

```text
PCADD
```

#### W2

```text
LIR
PCINC
```

含义：

```text
C = 0：不跳转，直接顺序取下一条指令
C = 1：执行 PC 相对跳转，然后取下一条指令
```

------

### Z = 1，跳转


```text
LIR
PCINC
```

含义：

```text
Z = 0：不跳转，顺序执行
Z = 1：执行 PC 相对跳转
```

# 四、扩展指令执行流程（在原指令集基础上补充）

以下指令均为在原流程图基础上的补充，与原指令分开列出。

## 12. 1011：MOV

### W1

```text
M
S = 1010
ABUS
DRW

LIR
PCINC
SHORT
```

含义：将源寄存器 Rs 的值经 ALU 直通（M=1，S=1010 实现 F=A）写回目的寄存器 Rd，同时取下一条指令。

------

## 13. 1100：CMP

### W1

```text
S = 0110
ABUS
LDZ
LDC

LIR
PCINC
SHORT
```

含义：执行 Rd−Rs 减法运算，**不写回寄存器（无 DRW）**，只更新 Z、C 标志，同时取下一条指令。供后续 JC/JZ 判断使用。

------

## 14. 1101：NOT

### W1

```text
M
S = 0000
ABUS
DRW
LDZ

LIR
PCINC
SHORT
```

含义：对寄存器 Rd 按位取反，结果经 ABUS 写回 Rd 并更新 Z 标志，同时取下一条指令。
采用 74181 正逻辑模式下 S=0000（实现 F=¬A）。注意 S=0100 在该模式下是 NAND而非取反。

------

## 15. 1111：DEC

### W1

```text
S = 1111
CIN
ABUS
DRW
LDZ
LDC

LIR
PCINC
SHORT
```

含义：执行自减运算 Rd←Rd−1（S=1111 配合 CIN=1），结果写回 Rd 并更新 Z、C 标志，同时取下一条指令。





# 补充

在 TEC-8 实验系统中，由于按下复位按钮 CLR 后，指令寄存器 IR 被复位为00H，因此，无论存储器中写入什么测试程序，在图 2.2 所示的流程图中，第 1条被执行的指令的代码总是 00H。我们规定，指令代码为 00H 的指令是条 nop 指令
